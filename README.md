# AISD Staff Intelligence Tool

A comprehensive public transparency tool for Austin Independent School District.

## What This Is
- 13,795 AISD employees with org chart, salaries, benefits, and contact info
- TEA 2024-25 enrollment data for 124 campuses
- FY2024-25 adopted budget drill-down (category → subcategory → line items)
- Vendor contracts and bid tabulations
- Full data source documentation

## Data Sources
All data is from public records:
- AISD OrgChartNow organizational chart
- AISD Staff Directory (live scrape)
- AISD 2025-26 Compensation Manual
- TEA PEIMS Enrollment Data (2024-25)
- AISD FY2024-25 Adopted Budget
- Travis CAD property values

## Built With
Single-file HTML app — no backend required, all data embedded.
